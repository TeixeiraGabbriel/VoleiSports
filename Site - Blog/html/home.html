Quem Somos



O blog VoleiSports é uma plataforma para os amantes do vôlei, aqui você tem acesso a todos os tipos de entretenimento, você também pode consultar acessórios esportivos do esporte.

Você encontrará no site notícias dos campeonatos, filmes ou desenhos relacionados, e jogos eletrônicos.
A VoleiSports é um site com tecnologias simples, utilizando programações em Javascript, CSS e HTML, ele foi feito com o intuito de facilitar o contato ao esporte.

O Projeto



Bem-vindo ao Site VoleiSports

O site foi criado para incentivar à prática ao esporte, de acordo com os objetivos de desenvolvimento sustentável da ONU, o esporte está ligado diretamente à saúde e bem-estar, além disso o esporte é inclusivo promovendo o esporte para deficientes fisícos.
